This project is intended first and foremost to give me the opprotunity to self teach some C/C++, electronics and hardware.
The problem it aims to satisfy is pretty niche, Slocum glider piloting, and will not be useful to many.
----
The Glidr Buddy desktop display aims to be a way to keep track of multile gliders operating on different schedules and in different conditions, freeing up monitor space and needing one less 
tab open, the buddy while be an "always on" heads up display showing several gliders (dependent on E-ink size and data retrieval method), the time since they last surfaced, time until next
surfacing, distance covered, as well as a depth averaged current flag. The included OLED screen will display time in UTC format, as a way to standardize operations across timezones.
